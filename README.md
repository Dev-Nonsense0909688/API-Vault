# API Vault

_**"A secure, self-hosted way to save, view, and use your API."**_

## Features:-

- Argon2 based password hashing
- JWT-based sessions
- Simple **web** dashboard
- REST API
- Fernet (AES-128) encrypted api-keys

## Project Structure

```
api-vault/
├── api/
├── core/
├── database/
├── middleware/
├── static/
├── templates/
└── app.py
```
